# RTL-to-GDSII ASIC Physical Design using OpenLane and Sky130

## Overview
This project demonstrates the complete ASIC Physical Design flow from RTL to GDSII using the OpenLane automated design flow and the Sky130 Process Design Kit (PDK). It covers all major stages of physical design, including synthesis, floorplanning, placement, clock tree synthesis (CTS), routing, and signoff.

## Objectives
- Learn the complete RTL-to-GDSII ASIC design flow.
- Understand each stage of the OpenLane flow.
- Generate a manufacturable GDSII layout.
- Analyze timing, power, area, and design rule checks.

## Tools Used
- OpenLane
- OpenROAD
- Magic VLSI
- Netgen
- KLayout
- Yosys
- Sky130 PDK

## Design Flow
1. RTL Design
2. Logic Synthesis
3. Floorplanning
4. Power Planning
5. Placement
6. Clock Tree Synthesis (CTS)
7. Routing
8. Static Timing Analysis (STA)
9. Design Rule Check (DRC)
10. Layout Versus Schematic (LVS)
11. GDSII Generation

## Features
